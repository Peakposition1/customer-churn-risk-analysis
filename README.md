# Customer Churn Risk & VIP Analysis (SQL + Python)

## 📊 Project Overview
This project identifies high-value customers at risk of churn using SQL, Python, and machine learning.

The goal is to help businesses prioritize retention efforts by analyzing:
- Customer value
- Purchase frequency
- Recency (last purchase)
- Predictive churn probability

## 🧠 Key Features
- SQL-based churn risk extraction
- Data cleaning and feature engineering
- Revenue at risk analysis
- Customer risk scoring model
- Logistic Regression churn prediction
- Data visualization with matplotlib

## 📈 Key Insights
- Customer inactivity is the strongest predictor of churn.
- High-value customers still churn if inactive.
- VIP customers were ranked by churn risk.
- Predictive model outputs churn probability.

## 🛠️ Tools Used
- SQL (SQLite)
- Python
- pandas
- matplotlib
- scikit-learn

## 📂 Files
- `customer_churn_project.ipynb` — main analysis notebook
